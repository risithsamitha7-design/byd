# Nissan GT-R Frontend

A stunning, modern frontend website showcasing the legendary Nissan GT-R supercar.

## Features

- Responsive design (mobile-friendly)
- Smooth scroll animations
- Modern glassmorphism UI
- GT-R specs and history
- Performance metrics
- Multiple GT-R model showcases

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript (intersection observers, parallax)

## Hosting

This site is configured for GitHub Pages. After pushing to the `main` branch, enable Pages in the repository settings with the source set to `main` branch / `/root` folder.
